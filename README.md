# manta-indexer

## sqlx-cli
- Install [sqlx-cli]() for creating db or migration.
```shell
cargo install sqlx-cli
```

## Deployment
- Create db
```shell
sqlx db create
```

- Run db migrations
```shell
sqlx migrate run
```

## Testing
- Create db
```shell
sqlx db create
```

- Run db migrations
```shell
sqlx migrate run
```

- Run tests
```shell
cargo t
```
